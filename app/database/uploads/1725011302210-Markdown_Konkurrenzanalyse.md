# Tools/Editoren

## Mark:
- Eine Ansicht auf einmal
	- STRG + K wechselt Ansicht
	- In Editoransicht Syntax immer angezeigt
- Gratis
- Web basiert
- Datei Speichern als:
	- HTML, PDF, IMAGE
- Opensource


## Holocron:
- Funktioniert nur am Desktop	
- Zwei Ansichten aufeinmal
	- Markdown syntax und Text
	- Man kann in beiden Ansichten schreiben
- Man kann andere Nutzer einladen
- Fileexplorer
	- Projekte
	- Ordner
	- Mehrere Files
- Free Trial, Anmeldung, Pro Version (ab 6$ pro Monat -	mehrere Hundert)
- Commands mit „/“
	- z.B. Tabellen
- Eigene Elemente
	- Callouts
	- Steps
	- Cards
	- Todo
- Spellchecker
- Cloud/GitHub Integration


## Minimalist Online Markdown Editor:
- Zwei Ansichten	
	- Markdown Editor und Text Preview
	- Preview kann auch als HTML angezeigt werden
	- Nur im Editor kann man schreiben
	- Kann zu einer Ansicht gewechselt werden (Fullscreein mode)
- Spellchecker
- Einstellungen
	- Nur Textgröße 
- Opensource


## StackEdit:
- Zwei Ansichten
	- Editor und Preview
	- Können beliebig ein und ausgeblendet werden
	- Nur im Editor kann geschrieben werden
- Fileexplorer
	- Ordner
	- Files
- Toolbar	
	- Undo/Redo
	- Alle Standard Markdown Optionen
- Menu
	- Google Account Integration
	- Workspaces
	- Cloud
	- Export
	- Drucken
	- Achievments/Badges
	- Templates
- Settings
	- Als YAML gespeichert
	- Aussehen und Keybinds können beliebig geändert werden
- Opensource – Spenden


## Dillinger:
- Zwei Ansichten
	- Editor und Preview
	- Können beliebig ein und ausgeblendet werden
	- Nur im Editor kann geschrieben werden
- Preview und Export als:
	- HTML
	- Styled html
	- Markdown
	- Pdf
- Cloud und GitHub kompatibel
- File Explorer
- Opensource
- Gratis


## MarkTwo:
- Eine Ansicht
	- Nur Preview Ansicht
	- Klickt man auf Text wechselt nur dieser Absatz zur Syntax
- „/“-Commands
	- Daten und Bilder
- Spellchecker
- „Fileexplorer“
	- Mehrere Dokumente
- Opensource
- Gratis


## Typo
- Zwei Ansichten
	- Nur in einer schreiben
	- Ansichten können getoggled werden
- Toolbar
- Formatier Button
	- Passt z.B. Tabellen an
- Präsentations Modus
- Gratis
- Opensource

## Dinugs:
- Rudimentärer Editor

- Drei Ansichten
	- Markdown Syntax
	- HTML
	- Text

- Convert-Button um Text zu konvertieren


### FAZIT:
Die meisten der webbasierten Tools ähneln sich sehr. Es gibt meist eine zwei Spalten Ansicht, wobei man nur in der Markdown-Ansicht editieren kann. Die Preview-Ansicht lässt ich in den meisten fällen zwischen HTML und Text ändern. Mark, MarkTwo und Holocron bilden hier die Ausnahme.
Die meistens tools waren opensource und somit auch gratis, auch hierbei bildet Holorcron wieder die Ausnahme.
Eine Toolbar und oder ein Commandsystem waren ebenso häufig vertreten, allerdings unterschiedlich gut und tiefgreifend implementiert.
Nur Holocron hat eigene Elemente angeboten, wie z.B. Cards oder Step-by-Step Anleitungen.
Typo bietet als einziges Tool einen Formatieren-Button, welcher vor allem bei der Verwendung von Tabellen nützlich war, da man so die Tabellen nicht per Hand an den Inhalt anpassen muss. 
Außerdem hat Typo als einziger einen Präsentationsmodus. Durch „–--„ markiert man das ende einer Folie, macht man dies nicht kann man die Folie scrollen.
